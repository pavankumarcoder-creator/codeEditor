# 🧑‍💻 Code Editor (HTML, CSS, JavaScript)

A simple in-browser code editor that lets you write HTML, CSS, and JavaScript and see the output instantly in a live preview window.

---

## 🚀 Features

- Write HTML, CSS, and JavaScript in separate panels
- Live preview using iframe
- Instant updates on every key press
- Simple and beginner-friendly UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome (for icons)

## ⚙️ How It Works

- The editor has three text areas:
  - HTML input
  - CSS input
  - JavaScript input
- On every key press (`onkeyup`), the `run()` function is triggered
- The function updates an `<iframe>` to show live output

---

## ▶️ How to Run the Project

1. Download or clone the repository
2. Open `index.html` in any browser
3. Start writing code in the editor panels
4. See live output on the right side
